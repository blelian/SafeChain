# Overview

SafeChain is a mobile-first cybersecurity assistant built with Next.js, Capacitor, Rust, FastAPI, and Hedera Hashgraph. It helps users secure their accounts, passwords, and browsing habits by combining AI-driven password scoring, behavioral anomaly detection, phishing detection, and immutable security logging on Hedera.  

The purpose of this software is to practice building full-stack applications, demonstrate proficiency in multiple technologies (frontend, backend, AI/ML, blockchain, mobile), and deliver a functional mobile app with security and privacy features.  


---

# Development Environment

- **Frontend:** Next.js 13 + TypeScript + TailwindCSS + Capacitor  
- **Backend:** Rust (Actix Web) microservice, Python FastAPI for AI orchestration  
- **AI / ML:** PyTorch, spaCy / Transformers  
- **Blockchain:** Hedera Hashgraph (SDKs & Mirror Node APIs)  
- **Database:** PostgreSQL + Redis  
- **Deployment / DevOps:** Docker, OCI Container Registry, Oracle Kubernetes Engine, GitHub Actions, NetworkPolicies, TLS, HPA  
- **Programming Languages:** TypeScript, Rust, Python  
- **Tools Used:** Visual Studio Code, Node.js / pnpm, Cargo, Python venv
